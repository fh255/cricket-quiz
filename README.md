[View the project here](<https://fh255.github.io/cricket-quiz/>)

This is an online quiz about Cricket. In this quiz, the user is asked to set a username to start the game. The quiz contains three questions about cricket. The feedback page will appear at the end of the questions, which will mention the user's correct and incorrect answers.

## User Story

- Enter user name and start quiz
- The username will be visible at the top, and the first question with three possible answers will be shown below the username.
- For the first question, only the "Next" button will be visible. After that, there will be one "Previous" button for going back to the previous question and one "Next" button for going to the next question.
- After answering three questions, the user will receive the overall score.

## Feature

### Username
* As the user enters the website, they are asked to enter a username.
* The user can enter any username.
* The characters may be string or numerical.
* The user enters the username and starts the game.
* A message appears on the screen: "Welcome to the quiz, [Username]."

## Deployment
- Go to GitHub, then select "Settings" from the right-side navigation.
- Select "Pages" from the left side.
- From the source section drop-down menu, choose the Main Branch.
- Once the Main branch is selected, the page will be automatically refreshed with a detailed ribbon display to indicate successful deployment.
